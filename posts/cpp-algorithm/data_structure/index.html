Program’s Memory: Heap and Stack

Variable Types Storage

Primitive: int x = 7;

  • name: x
  • value: 7
  • address: 0x12AB5
  • size: 4 bytes(int)

Reference: Circle c; c = new Circle(7);

  • name: c -> name: c
  • address: 0x4AF23 -> address: 0x3EF3B(new memory address)
  • size: ? -> size: address: 8 bytes (64 bit)
  • value: ? (better to assign null) -> value: 0x3EF3B(store the memory address, or points to)
  • If no pointer points to the object, the object will turn into Garbage Collection

Memory Storage

Stack

  • Stack frames: collection of memory blocks(ONLY create while functions runs)
  • Size: could be varied concerning variables, number of functions
  • content: the function arguments, all locally created variables, the return value

Heap

  • Stack: Circle c;

  • Heap: c = new Circle();

  • Array on the heap: int[] array = new int[7];

    • Current_address = initial_address + index * size_of_type

How to create a data structure?

A List ADT

  • (ADT) Abstract Data Type

    • Intuitive list

      • a collection of elements with some notion of position information for each element
      • operations:
        • add/remove at specified positions
        • find and element in the list (or determine that it is not there)
        • retrieve and element from a particular position
        • determine the length of the list
        • empty the list

    • List ADT

      • values stored
        • the list of elements itself
        • size - current number of elements in the list
      • operations
        • add(item, pos) - adds an element at position pos, restriction: 0 <= pos < size, shifts elements starting at pos one to the right (higher position values)
        • remove(pos) - removes and returns an element at position pos, restriction: 0 <= pos < size, shifts elements starting at pos+1 one to the left (lower position values)
        • find(item) - determines if the item is in the list and if so returns its position, otherwise returns -1
        • get(pos) - retrieves and returns an item from position pos, restriction: 0 <= pos < size
        • size() - returns the current number of elements in the list, i.e. size
        • clear() - removes all elements from the list, and reset size to 0

    • Empty List: Node n = new Node(); head = n, tail = n;
    • Push Front: Node n = new Node(); n.next = head; head = n;
