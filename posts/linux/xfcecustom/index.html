Linux

Xfce customization and applications

06/21/23
1 minute read

System

  • Gnome Keyring: gnome-keyring libsecret libgnome-keyring

  • Power: tlp tlp-rdw

  • Network: network-manager-applet networkmanager-openconnect (because of some bug, 1.2.8 is recommended)

  • FTP: gvfs

  • AppImage: AUR - appimagelauncher

  • compress: file-roller thunar-archive-plugin

  • AUR manager: yay

    1. download snapshot
    2. $ makepkg
    3. $ sudo pacman -U *.pkg.tar.zst

Appearance

  • HiDPI: Application - Settings - Appearance : DPI 128

  • fonts

    • packages
      • ttf-ubuntu-font-family
      • ttf-fira-code
      • noto-fonts noto-fonts-cjk noto-fonts-emoji
    • Default Font: Ubuntu Regular 10.5
    • Default Mono Font: Fira Code Regular 10.5

  • icons: Tela icons

  • terminal: xfce4-terminal --drop-down

    • .bashrc: [\u@\h \W]\$ -> [\u]\w\$

  • locker & login

    • /etc/lightdm/lightdm-gtk-greeter.conf:

      ```conf
[greeter]
background=/usr/share/backgrounds/xfce/bg.png
#user-background=
theme-name=Adwaita-dark
icon-theme-name=Tela-ubuntu-dark
font-name=Ubuntu 10.5
xft-antialias=1
xft-dpi=128
xft-hintstyle=hintfull
xft-rgba=none
```

  • Desktop: conky

Media

  • Sound

    • Basis: pipewire-pulse pipewire-alsa pipewire-jack pipewire alsa-utils
    • GUI: xfce4-pulseaudio-plugin
    • Mix: pavucontrol
    • Noise suppression: noise-suppression-for-voice
    • Music Player: audacious (Status Icon in Config)

  • Video

    • Video Player: vlc

  • Image

    • Screenshot: flameshot

Input

  • Method: fcitx5

    • Basis: fcitx5-im fcitx5-chinese-addons
    • GUI: fcitx5-nord

  • Clipboard: xfce4-clipman-plugin

  • Dictionary: goldendict

Work

  • Code: AUR - visual-studio-code-bin this package contains full functions of vs-code including sync with GitHub. (Need Gnome Keyring above)

    • manager: git

  • Paper: AUR - zotero

    • Better Bibtex

  • libreoffice-still

  • email: evolution

    • System tray: evolution-on

  • community: telegram-desktop

  • Browser: firefox

Game

  • Steam: steam

    • enable multilib: /etc/pacman.conf - [multilib] - uncomment Include = /etc/pacman.d/mirrorlist
      • refresh: $ sudo pacman -Syy

  • osu!: osu!
