Better BibTex for Zotero

  • BibTex Style as Google Research

    • Edit -> preference -> Better BibTex
    • Citation key formula: auth.lower + year + shorttitle(1,0).lower

  • Copy BibTex

    • Edit -> preference -> Export
    • Item Format: Better BibTex

  • Exclude(omit) unnecessary item

    • Better BibTex -> Export -> Fields
    • Fields to omit: abstract, urldate, file, annotation, langid
